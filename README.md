# Persona Chat Frontend

A React + Tailwind chat UI for a coding assistant persona.

## Features

- Clean chat interface with animated background
- Light and dark mode toggle
- Typing indicator and auto-scroll to latest message
- Suggested topics in empty state
- Backend chat API integration

## Tech Stack

- React 18
- Tailwind CSS
- Create React App (`react-scripts`)

## Getting Started

### Prerequisites

- Node.js 16+ (recommended)
- npm

### Install

```bash
npm install
```

### Run in Development

```bash
npm start
```

App runs at `http://localhost:3000`.

### Build for Production

```bash
npm run build
```
